# Calculator Demo
- - -

## Quick Start

1. Clone the repository
```bash
git clone https://github.com/adam-r-kowalski/calculator.git
```

2. Start a repl
```bash
lein repl
```

3. Start figwheel
```cljs
(start)
```

4. Start the clojurescript repl
```cljs
(cljs)
```

5. Point your browser to: ```http://localhost:3449```
